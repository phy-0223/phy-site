# phy · 个人门户

极简暗色风格的个人展示页，部署在 GitHub Pages。

## 在线访问

**https://phy-0223.github.io/phy-site/**

## 技术栈

- HTML5  
- [Tailwind CSS](https://tailwindcss.com/)（CDN）

## 仓库结构

| 文件 | 说明 |
|------|------|
| `index.html` | 单页站点（导航、自我介绍、技术栈、作品、联系与表单） |
| `me.png` | 头像资源（需放在仓库根目录，供 Pages 正确加载） |

## 本地预览

直接用浏览器打开 `index.html`，或使用任意静态服务器，例如：

```bash
npx serve .
```

## 部署说明

本仓库通过 **GitHub Pages**（`main` 分支根目录）发布。修改后推送至 `main`，等待片刻即可生效。
